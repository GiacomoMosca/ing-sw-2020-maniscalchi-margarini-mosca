## Instructions

### Server

`java -jar ServerApp.jar`

### Client

Two separate client JARs were built to easily choose between CLI and GUI interfaces.

For CLI:
`java -jar ClientCLI.jar`

For GUI:
`java -jar ClientGUI.jar`
